## Description

*Welcome back to Learn With Mithran! Today we kick off Phase 2 — Networking — but before we touch a single Azure networking service, we need to master the language every network speaks: IP addresses, binary, and CIDR notation.*

In this 100% conceptual, 100% free session, we break down exactly what an IP address is, how bits, bytes, and binary work under the hood, the difference between public and private IPs (including the three reserved ranges every Azure VNet uses), and how CIDR notation lets you calculate the size of any network block in seconds — plus a quick look at the Azure Portal's live CIDR calculator. By the end, a block like `10.0.1.0/24` will never look like a random string of numbers again, setting you up perfectly for Day 10, where we build your first Azure Virtual Network from scratch. 🚀
