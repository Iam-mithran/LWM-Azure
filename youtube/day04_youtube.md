## Description

*Welcome back to Learn With Mithran! Today we step back from clicking buttons and actually understand what we're building — then we build it. By the end of this session you'll know exactly how websites work under the hood, what 3-Tier Architecture means, and you'll have hosted a real webpage on both a Windows VM (IIS) and an Ubuntu VM (Nginx) in Azure.*

In this session we cover static vs dynamic websites, frontend vs backend vs database, architecture tiers (1-Tier, 2-Tier, 3-Tier), what a web server actually does, and two full portal demos — one on Windows Server with IIS and one on Ubuntu with Nginx. This is the foundation for everything we build in the capstone project. 🚀
