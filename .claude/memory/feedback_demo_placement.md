Demos must be placed immediately after each concept section — topic then demo, topic then demo — throughout the script. Never group all demos into a single "Portal Demo" section at the end of the file.

**Why:** The course structure per CLAUDE.md is: concept explanation → hands-on portal demo → next concept → hands-on portal demo. Grouping demos at the end breaks the teaching flow and makes it harder for students to follow along in real time.

**How to apply:** Every time a concept section ends, immediately follow it with the relevant portal steps (with ✅/💳 tier labels). Only skip a demo inline if there is a documented reason (e.g., "full demo in Day X with prerequisite service").
